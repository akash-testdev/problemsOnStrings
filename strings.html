Strings

This page contains problems on strings from geeks for geeks and their solutions.

Logic:-
Split the input string using the split method and have the words in a string array. Now maintain two poinnters, left pointer and right pointer pointing to the first and last element of the array. Till left pointer is less than the right pointer keep swapping the array elements. Finally traverse the array from the first element till the last, and concatenate each array element into a string and return the string.
Space complexity:- TBD
Time complexity:- O(n)

Note:-
1. The time complexity of split() method is O(n) where n is the length of the string.
2. In case the delimiter mentioned while calling split() method is not present in the string, then the array returned has just one element containing the entire input string. 

public String reverseWordsInString(String str)
{
  String s[] = str.split(" ");   
  int leftPointer = 0;
  int rightPointer = s.length-1;
  while(leftPointer<rightPointer)
  {
    String temp = s[leftPointer];
    s[leftPointer] = s[rightPointer];
    s[rightPointer] = temp;
    leftPointer++;
    rightPointer--;
  }
  leftPointer = 0;
  rightPointer = s.length-1;
  String reversedString = "";
  while(leftPointer<=rightPointer)
  {
    reversedString = reversedString+s[leftPointer]+" ";
    leftPointer++;
  }
  return reversedString.trim();
}
